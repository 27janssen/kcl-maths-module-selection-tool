What is this site? About

The KCL Mathematics Module Hub is a student-made resource for navigating the specification of KCL's Mathematics Courses. It provides an interactive prerequisite map for each degree within the department, a full browsable module list, and individual pages for every module with descriptions, assessment details, lecture notes, and past papers.

Use the Module Selection tool to visualise which modules unlock which — click any module card to highlight its prerequisites (green/blue) and the modules it unlocks (red). Double-clicking a module allows you to view a dedicated page for the module.
